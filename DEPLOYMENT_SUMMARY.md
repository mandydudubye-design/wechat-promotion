## 📋 你的问题

---

## 💡 为什么需要云部署？

### 本地开发 vs 生产环境

| 对比项 | 本地开发 | 云服务器 |
|--------|---------|---------|
| **访问** | 只有你能访问 | 所有人都能访问 |
| **稳定性** | 电脑关机就停 | 24小时运行 |
| **微信回调** | 无法配置 | 必须有公网域名 |
| **数据安全** | 电脑坏了数据丢失 | 云数据库自动备份 |
| **扩展性** | 无法扩展 | 可以随时升级配置 |

### 微信公众号要求

```
微信公众号必须要求：
1. 服务器地址：公网可访问的域名
2. HTTPS协议：必须SSL证书
3. 响应时间：< 5秒
4. 在线率：99.9%以上

本地开发环境无法满足这些要求！
```

## 🎯 部署方案选择

### 方案A：云服务器 + 云数据库（推荐⭐）

**适合**：正式运营、100人以上

**优点**：
- ✅ 性能好、稳定性高
- ✅ 数据自动备份
- ✅ 可以随时升级
- ✅ 专业运维支持

**缺点**：
- ❌ 成本稍高（¥110-180/月）
- ❌ 需要配置两次

### 方案B：轻量应用服务器

**适合**：测试、小规模（<50人）

**优点**：
- ✅ 便宜（¥100-150/月）
- ✅ 配置简单

**缺点**：
- ❌ 性能一般
- ❌ 数据备份麻烦
- ❌ 扩展性差

### 方案C：免费方案（不推荐）

**方案**：
- 使用本地服务器（如：公司内网）
- 使用内网穿透（如：frp、ngrok）

**缺点**：
- ❌ 不稳定
- ❌ 速度慢
- ❌ 微信审核不通过
- ❌ 数据安全无保障

## 🚀 快速部署（3种方式）

### 方式1：自动部署（推荐）

```bash
# 1. 上传deploy.sh到服务器
scp deploy.sh root@your-server-ip:/root/

# 2. 连接服务器
ssh root@your-server-ip

# 3. 给脚本执行权限
chmod +x deploy.sh

# 4. 运行自动部署脚本
./deploy.sh

# 脚本会自动完成：
# - 系统更新
# - 安装Node.js、Nginx、PM2
# - 配置数据库
# - 部署后端代码
# - 配置Nginx
# - 申请SSL证书
```

**时间：30分钟**

### 方式2：手动部署

```bash
# 按照DEPLOYMENT_CHECKLIST.md逐步操作
# 每一步都有详细说明和验证
```

**时间：1-2小时**

### 方式3：使用Docker（最快）

```bash
# 1. 安装Docker
curl -fsSL https://get.docker.com | sh

# 2. 运行容器
docker run -d \
  --name wechat-promotion \
  -p 3000:3000 \
  -e DB_HOST=your-db-host \
  -e DB_USER=your-db-user \
  -e DB_PASSWORD=your-db-password \
  your-docker-image

# 3. 配置Nginx反向代理
```

**时间：15分钟**

## 🔧 运维管理

### 日常操作

```bash
# 查看后端状态
pm2 status

# 查看后端日志
pm2 logs wechat-backend

# 重启后端
pm2 restart wechat-backend

# 查看Nginx日志
tail -f /var/log/nginx/wechat-promotion-access.log

# 重启Nginx
systemctl restart nginx
```

### 数据库备份

```bash
# 手动备份
mysqldump -h db-host -u user -p dbname > backup.sql

# 自动备份（每天凌晨3点）
0 3 * * * /var/www/scripts/backup-db.sh
```

### 监控告警

```bash
# 安装监控
pm2 install pm2-logrotate

# 查看监控
pm2 monit
```

## 📞 技术支持

### 常见问题

**Q1：服务器选择阿里云还是腾讯云？**
- 推荐：阿里云（稳定性好、文档全）
- 备选：腾讯云（价格稍低）

**Q2：必须买云数据库吗？**
- 推荐：是（自动备份、高可用）
- 备选：服务器自建MySQL（省钱但风险高）

**Q3：必须买域名吗？**
- 推荐：是（微信要求、专业）
- 备选：使用IP地址（不稳定）

**Q4：SSL证书多少钱？**
- 免费：Let's Encrypt（推荐）
- 付费：阿里云/Tencent云（¥500-3000/年）
