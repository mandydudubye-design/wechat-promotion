## 🎯 核心功能

### 1. 公众号类型选择

在添加或编辑公众号时，可以选择公众号类型：
- **订阅号**（subscription）：通过识别码验证
- **服务号**（service）：通过H5网页授权

### 2. 自动绑定流程

系统根据公众号类型自动选择绑定方式：

**服务号**：
- H5网页授权（静默授权）
- 用户无感知
- 体验最佳

**订阅号**：
- 识别码验证
- 员工在公众号对话框发送：`工号 + 手机号后4位`
- 体验稍差，但功能完整

### 3. 新增 API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/init/migrate-account-type` | POST | 执行数据库迁移 |
| `/api/employee-unified/bind-config` | GET | 获取公众号绑定配置 |
| `/api/employee-unified/bind` | POST | 统一员工绑定接口 |
| `/api/verification-code/send` | POST | 发送识别码（订阅号） |
| `/api/verification-code/verify` | POST | 验证识别码（订阅号） |
| `/api/wechat-callback/unified` | POST | 微信统一回调处理 |
| `/api/wechat-oauth/auth-url` | GET | 获取微信授权 URL |
| `/api/wechat-oauth/callback` | GET | 处理微信授权回调 |

## 🐛 常见问题

### 1. 数据库迁移失败

**问题**：执行迁移时提示"字段已存在"

**解决**：这是正常的，表示字段已经存在，可以忽略。

### 2. 微信授权失败

**问题**：服务号绑定时提示"授权失败"

**解决**：
- 检查 AppID 和 AppSecret 是否正确
- 检查网页授权域名是否配置（微信公众平台 → 开发 → 接口权限 → 网页授权）
- 必须在微信内打开链接（浏览器打开会失败）

### 3. 识别码验证失败

**问题**：订阅号绑定时提示"识别码错误"

**解决**：
- 检查识别码格式是否正确（工号 + 手机号后4位）
- 检查员工信息是否正确
- 确保识别码是在公众号对话框中发送的

### 4. 后端启动失败

**问题**：提示"数据库连接失败"

**解决**：
- 检查 MySQL 服务是否启动
- 检查 `.env` 文件中的数据库配置是否正确
- 确认数据库 `wechat_promotion` 是否存在
