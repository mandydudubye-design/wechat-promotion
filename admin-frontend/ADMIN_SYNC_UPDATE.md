# 管理后台同步调整说明

## 📋 概述

在员工端H5完成多公众号UI优化的同时，对管理后台进行了同步调整，新增了多公众号管理功能页面。

## ✨ 新增功能

### 1. 多公众号管理页面

#### 页面路径
- 路由：`/accounts`
- 组件：`AccountsPage.tsx`
- 位置：`src/components/pages/AccountsPage.tsx`

#### 核心功能

##### 📊 统计卡片
```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│ 公众号总数  │ 活跃公众号  │ 推广员工数  │ 总扫码数    │
│     3       │      2      │    156      │   12,345    │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

##### 📝 公众号列表
显示所有已配置的公众号，包括：
- 公众号头像和名称
- 状态标签（活跃/停用）
- 描述信息
- AppID（支持一键复制）
- 操作按钮（启用/停用、编辑、删除）

##### ➕ 添加公众号
通过对话框添加新的公众号：
- 公众号名称
- AppID
- AppSecret
- 描述信息

##### ✏️ 编辑公众号
修改现有公众号的配置信息

##### 🔄 状态管理
- 启用/停用公众号
- 删除公众号（需确认）

## 🎯 使用指南

### 添加公众号
1. 打开管理后台
2. 导航到"公众号管理"
3. 点击"添加公众号"按钮
4. 填写公众号信息
5. 点击"添加"完成

### 管理公众号
1. 在公众号列表找到目标公众号
2. 使用右侧操作按钮：
   - **启用/停用**：控制是否可推广
   - **编辑**：修改公众号信息
   - **删除**：删除公众号（需确认）

### 复制AppID
1. 在公众号列表找到目标公众号
2. 点击AppID旁边的复制按钮
3. AppID已复制到剪贴板
