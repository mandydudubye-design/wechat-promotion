# 员工端H5 - 公众号推广追踪系统

## 项目简介

这是公众号推广追踪系统的员工端H5移动应用，供组织内部员工使用，可以查看专属推广码、推广数据和排行榜。

## 技术栈

- **React 19** - UI框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **React Router** - 路由管理
- **Ant Design Mobile** - 移动端UI组件库
- **QRCode.react** - 二维码生成

## 功能特性

### 1. 账号绑定
- 扫码进入绑定页面
- 输入工号、姓名、手机号验证身份
- 完成微信账号绑定

### 2. 我的推广码
- 查看专属推广二维码
- 实时推广数据统计
  - 累计扫码数、关注数
  - 今日扫码数、关注数
  - 本月扫码数、关注数
  - 转化率计算
- 复制推广文案
- 保存二维码图片

### 3. 推广榜单
- 今日排行榜
- 本月排行榜
- 查看我的排名
- 实时数据更新

### 4. 推广记录
- 查看所有推广记录
- 显示关注状态（已关注/未关注）
- 显示用户昵称和关注时间
- 按时间倒序排列

### 5. 个人中心
- 查看个人信息
- 推广数据统计
- 快捷功能入口

### 6. 帮助说明
- 快速入门指南
- 常见问题解答
- 联系方式

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```
访问 http://localhost:5176

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 开发规范

### 目录结构
```
src/
├── pages/          # 页面组件
├── components/     # 公共组件
├── utils/          # 工具函数
├── types/          # 类型定义
├── services/       # API服务
└── hooks/          # 自定义Hooks
```

### 命名规范
- 组件文件：PascalCase（如 HomePage.tsx）
- 样式文件：PascalCase.css（如 HomePage.css）
- 工具函数：camelCase（如 formatDateTime）
