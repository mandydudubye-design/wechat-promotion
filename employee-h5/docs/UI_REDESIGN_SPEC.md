## 🧩 组件设计

### 1. 统计卡片（优化版）

#### 设计要点
- **层次清晰**：标题 → 数据 → 辅助信息
- **视觉焦点**：数据突出显示
- **微交互**：点击反馈、悬停效果

#### 布局结构
```
┌─────────────────────────────────┐
│  🏆 推广数据          排名 #5   │
│  ├─────────────────────────────┤
│  │  📊 1,234      👥 456        │
│  │  扫码数        关注数        │
│  ├─────────────────────────────┤
│  │  📈 37.5%                    │
│  │  关注率                      │
│  └─────────────────────────────┘
│  今日 +89 | 本周 +234           │
└─────────────────────────────────┘
```

#### 样式规格
- 背景：渐变 #667eea → #764ba2
- 文字：白色
- 圆角：16px
- 内边距：20px
- 阴影：shadow-lg

### 2. 公众号选择器（优化版）

#### 设计要点
- **清晰标识**：图标 + 名称 + 描述
- **状态反馈**：选中状态明显
- **流畅交互**：展开/收起动画

#### 布局结构
```

│  📱 选择公众号           ▼     │
│  ┌─────────────────────────────┐│
│  │ 🏢 企业服务号          ✓   ││
```

## 🎬 动画系统

### 过渡效果

#### 标准过渡
```css
--transition-fast: 150ms cubic-bezier(0.4, 0, 0.2, 1);
--transition-base: 200ms cubic-bezier(0.4, 0, 0.2, 1);
--transition-slow: 300ms cubic-bezier(0.4, 0, 0.2, 1);
```

#### 弹性过渡
```css
--transition-bounce: 500ms cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 动画效果

#### 淡入淡出
```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
```

#### 滑入
```css
@keyframes slideInUp {
  from {
    transform: translateY(20px);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}
```

#### 缩放
```css
@keyframes scaleIn {
  from {
    transform: scale(0.9);
    opacity: 0;
  }
  to {
    transform: scale(1);
    opacity: 1;
  }
}
```

## 🎯 设计原则

### 1. 清晰层次
- 使用大小、颜色、间距建立视觉层次
- 重要信息突出显示
- 次要信息弱化处理

### 2. 一致性
- 统一的颜色使用
- 统一的间距系统
- 统一的交互模式

### 3. 反馈
- 即时的点击反馈
- 明确的状态变化
- 流畅的动画过渡

### 4. 可访问性
- 足够的颜色对比度
- 清晰的文字大小
- 明确的交互提示
