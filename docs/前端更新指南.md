## 二、统一员工绑定页面

### 2.1 创建统一绑定页面

已创建文件：`employee-h5/src/pages/EmployeeUnifiedBindPage.tsx`

这个页面会根据公众号类型自动选择绑定方式：
- **服务号**：自动进入H5授权流程
- **订阅号**：提示用户先发送识别码，然后进入验证流程

### 2.2 使用方式

**服务号绑定**：
```
https://your-domain.com/bind?account_id=1
```

**订阅号绑定**：
```
https://your-domain.com/bind?account_id=2
```

### 3.1 后端路由注册

在 `backend/src/index.ts` 中注册新路由：

```typescript
import employeeUnifiedRoutes from './routes/employee-unified';
import verificationCodeRoutes from './routes/verification-code';

// 注册路由
app.use('/api/employee', employeeUnifiedRoutes);
app.use('/api/verification-code', verificationCodeRoutes);
app.use('/api/wechat-callback', wechatCallbackUnifiedRoutes);
```

### 3.2 前端API更新

在 `frontend/src/lib/api.ts` 中添加：

```typescript
// 获取公众号绑定配置
export async function getAccountBindConfig(accountId: number) {
  const response = await fetch(`/api/employee/bind-config?account_id=${accountId}`, {
    credentials: 'include',
  });
  return response.json();
}

// 统一员工绑定
export async function bindEmployeeUnified(data: {
  account_id: number;
  employee_id: string;
  phone_suffix: string;
  openid?: string;
  bind_method?: string;
}) {
  const response = await fetch('/api/employee/bind', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    credentials: 'include',
    body: JSON.stringify(data),
  });
  return response.json();
}

// 获取员工绑定状态
export async function getEmployeeBindStatus(employeeId: string, accountId?: number) {
  const params = new URLSearchParams({ employee_id });
  if (accountId) params.append('account_id', accountId.toString());

  const response = await fetch(`/api/employee/bind-status?${params}`, {
    credentials: 'include',
  });
  return response.json();
}

// 验证识别码
export async function verifyCode(data: {
  account_id: number;
  openid: string;
  content: string;
}) {
  const response = await fetch('/api/verification-code/verify', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    credentials: 'include',
    body: JSON.stringify(data),
  });
  return response.json();
}

// 生成验证码（短信）
export async function generateVerificationCode(data: {
  employee_id: string;
  phone: string;
}) {
  const response = await fetch('/api/verification-code/generate', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    credentials: 'include',
    body: JSON.stringify(data),
  });
  return response.json();
}

// 验证短信验证码
export async function validateSmsCode(data: {
  phone: string;
  code: string;
}) {
  const response = await fetch('/api/verification-code/validate', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    credentials: 'include',
    body: JSON.stringify(data),
  });
  return response.json();
}
```

## 五、测试流程

### 5.1 测试服务号

1. **添加服务号**
   - 进入公众号管理页面
   - 点击"添加公众号"
   - 选择"服务号"
   - 填写公众号信息

2. **生成绑定链接**
   - 复制服务号的ID
   - 构造链接：`https://your-domain.com/bind?account_id=1`

3. **员工绑定**
   - 在微信内打开链接
   - 系统自动完成H5授权
   - 输入工号+手机号后4位
   - 验证成功

### 5.2 测试订阅号

1. **添加订阅号**
   - 进入公众号管理页面
   - 点击"添加公众号"
   - 选择"订阅号"
   - 填写公众号信息

2. **生成绑定链接**
   - 复制订阅号的ID
   - 构造链接：`https://your-domain.com/bind?account_id=2`

3. **员工绑定**
   - 在公众号对话框发送识别码：`EMP0011234`
   - 在微信内打开绑定链接
   - 输入工号+手机号后4位
   - 验证成功

## 七、注意事项

1. **域名配置**
   - 服务号需要在微信后台配置"网页授权域名"
   - 不要带 http:// 或 https://
   - 例如：`your-domain.com`

2. **微信回调**
   - 确保微信服务器能访问你的回调地址
   - 需要在微信后台配置服务器地址
   - 格式：`https://your-domain.com/api/wechat-callback/callback`

3. **数据库字段**
   - 确保执行了数据库迁移脚本
   - 检查 `wechat_accounts` 表是否有 `account_type` 字段

4. **OpenID获取**
   - 服务号通过H5网页授权获取
   - 订阅号通过识别码消息获取

5. **测试环境**
   - 可以使用微信测试号进行测试
   - 测试号支持H5网页授权
