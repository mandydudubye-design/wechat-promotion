# 订阅号/服务号双模式架构设计

## 一、架构概览

### 1.1 设计原则

```
┌─────────────────────────────────────────────────────────────┐
│                    双模式统一架构                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌──────────────┐         ┌──────────────┐                 │
│  │   服务号     │         │   订阅号     │                 │
│  │  Service     │         │  Subscription│                 │
│  └──────┬───────┘         └──────┬───────┘                 │
│         │                        │                           │
│         │ H5网页授权              │ 识别码方案                 │
│         │ (自动获取OpenID)        │ (员工主动发送)             │
│         │                        │                           │
│         └────────────┬───────────┘                           │
│                      │                                       │
│                      ↓                                       │
│         ┌────────────────────────┐                          │
│         │   统一绑定接口层        │                          │
│         │  (根据类型自动路由)     │                          │
│         └──────────┬─────────────┘                          │
│                      │                                       │
│                      ↓                                       │
│         ┌────────────────────────┐                          │
│         │   员工绑定统一流程      │                          │
│         └────────────────────────┘                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 核心特性

✅ **统一入口**：员工通过同一个H5页面绑定
✅ **自动识别**：根据公众号类型自动选择绑定方案
✅ **无缝切换**：支持同时管理多个不同类型的公众号
✅ **用户无感**：服务号使用静默授权，订阅号使用识别码

## 三、后端设计

### 3.1 公众号类型判断服务

```typescript
// backend/src/services/accountTypeService.ts

export enum AccountType {
  SUBSCRIPTION = 'subscription', // 订阅号
  SERVICE = 'service'           // 服务号
}

export class AccountTypeService {
  /**
   * 获取公众号类型
   */
  async getAccountType(accountId: number): Promise<AccountType> {
    const [accounts] = await pool.query(
      'SELECT account_type FROM wechat_accounts WHERE id = ?',
      [accountId]
    );

    if (accounts.length === 0) {
      throw new Error('公众号不存在');
    }

    return accounts[0].account_type as AccountType;
  }

  /**
   * 判断是否支持H5网页授权
   */
  isWebAuthSupported(accountType: AccountType): boolean {
    return accountType === AccountType.SERVICE;
  }

  /**
   * 获取推荐绑定方式
   */
  getRecommendedBindMethod(accountType: AccountType): 'web_auth' | 'verification_code' {
    return accountType === AccountType.SERVICE ? 'web_auth' : 'verification_code';
  }

  /**
   * 获取绑定方式描述
   */
  getBindMethodDescription(accountType: AccountType): string {
    if (accountType === AccountType.SERVICE) {
      return 'H5网页授权（自动获取，无需操作）';
    } else {
      return '识别码验证（在公众号对话框发送识别码）';
    }
  }
}
```

### 3.2 统一员工绑定接口

```typescript
// backend/src/routes/employee-unified.ts

import { Router, Request, Response } from 'express';
import { AccountTypeService, AccountType } from '../services/accountTypeService';
import { pool } from '../config/database';
import { logger } from '../utils/logger';

const router = Router();
const accountTypeService = new AccountTypeService();

/**
 * 获取公众号绑定配置
 *
 * GET /api/employee/bind-config?account_id=1
 *
 * 返回该公众号支持的绑定方式
 */
router.get('/bind-config', async (req: Request, res: Response) => {
  try {
    const accountId = parseInt(req.query.account_id as string);

    if (!accountId) {
      return res.status(400).json({
        success: false,
        message: '缺少account_id参数'
      });
    }

    // 获取公众号信息
    const [accounts] = await pool.query(
      'SELECT id, account_name, account_type, app_id FROM wechat_accounts WHERE id = ? AND status = 1',
      [accountId]
    );

    if (accounts.length === 0) {
      return res.status(404).json({
        success: false,
        message: '公众号不存在或已停用'
      });
    }

    const account = accounts[0];
    const accountType = account.account_type as AccountType;

    // 根据类型返回不同的配置
    const bindMethod = accountTypeService.getRecommendedBindMethod(accountType);
    const isWebAuthSupported = accountTypeService.isWebAuthSupported(accountType);

    res.json({
      success: true,
      data: {
        account_id: account.id,
        account_name: account.account_name,
        account_type: accountType,
        app_id: account.app_id, // 服务号才有
        bind_method: bindMethod,
        bind_method_description: accountTypeService.getBindMethodDescription(accountType),
        is_web_auth_supported: isWebAuthSupported,
        features: {
          web_auth: isWebAuthSupported,              // H5网页授权
          verification_code: true,                  // 识别码验证（两种类型都支持）
          auto_get_openid: isWebAuthSupported,      // 自动获取OpenID
          manual_send_code: true,                   // 手动发送识别码
        }
      }
    });

  } catch (error: any) {
    logger.error('获取绑定配置失败', error);
    res.status(500).json({
      success: false,
      message: error.message || '获取绑定配置失败'
    });
  }
});

/**
 * 统一员工绑定接口
 *
 * POST /api/employee/bind
 *
 * 根据公众号类型自动选择绑定方式
 */
router.post('/bind', async (req: Request, res: Response) => {
  try {
    const { account_id, employee_id, phone_suffix, openid, bind_method } = req.body;

    if (!account_id || !employee_id || !phone_suffix) {
      return res.status(400).json({
        success: false,
        message: '缺少必要参数'
      });
    }

    // 获取公众号类型
    const accountType = await accountTypeService.getAccountType(account_id);

    // 根据公众号类型选择绑定方式
    if (accountType === AccountType.SERVICE && !openid) {
      // 服务号必须提供openid（通过H5网页授权获取）
      return res.status(400).json({
        success: false,
        message: '服务号需要先进行H5网页授权获取OpenID'
      });
    }

    if (accountType === AccountType.SUBSCRIPTION && !openid) {
      // 订阅号没有openid，需要通过识别码绑定
      return res.status(400).json({
        success: false,
        message: '订阅号需要先在公众号对话框发送识别码'
      });
    }

    // 验证员工信息
    const [employees] = await pool.query(
      'SELECT * FROM employees WHERE employee_id = ? AND phone LIKE ?',
      [employee_id, `%${phone_suffix}`]
    );

    if (employees.length === 0) {
      return res.status(400).json({
        success: false,
        message: '工号或手机号后4位不正确'
      });
    }

    const employee = employees[0];

    // 检查是否已绑定
    const [existing] = await pool.query(
      'SELECT * FROM employee_bindings WHERE employee_id = ? AND account_id = ?',
      [employee_id, account_id]
    );

    if (existing.length > 0) {
      return res.status(400).json({
        success: false,
        message: '该员工已绑定此公众号'
      });
    }

    // 绑定员工
    await pool.query(
      `INSERT INTO employee_bindings (employee_id, account_id, openid, is_verified, verified_at, created_at)
       VALUES (?, ?, ?, TRUE, NOW(), NOW())`,
      [employee_id, account_id, openid || null]
    );

    // 生成推广二维码
    const sceneStr = `emp_${employee_id}_${Date.now()}`;
    // ... 生成二维码逻辑

    res.json({
      success: true,
      message: '绑定成功',
      data: {
        employee_id: employee.employee_id,
        employee_name: employee.name,
        account_id: account_id,
        bind_method: accountType === AccountType.SERVICE ? 'web_auth' : 'verification_code',
        qr_code_url: 'https://...' // 推广二维码URL
      }
    });

  } catch (error: any) {
    logger.error('员工绑定失败', error);
    res.status(500).json({
      success: false,
      message: error.message || '绑定失败'
    });
  }
});

export default router;
```

### 3.3 识别码处理（订阅号专用）

```typescript
// backend/src/routes/verification-code.ts

import { Router, Request, Response } from 'express';
import { pool } from '../config/database';
import { logger } from '../utils/logger';
import { AccountTypeService, AccountType } from '../services/accountTypeService';

const router = Router();
const accountTypeService = new AccountTypeService();

/**
 * 处理识别码消息
 *
 * POST /api/verification-code/verify
 *
 * 当用户在公众号对话框发送识别码时调用
 */
router.post('/verify', async (req: Request, res: Response) => {
  try {
    const { account_id, openid, content } = req.body;

    if (!account_id || !openid || !content) {
      return res.status(400).json({
        success: false,
        message: '缺少必要参数'
      });
    }

    // 验证是否是订阅号
    const accountType = await accountTypeService.getAccountType(account_id);
    if (accountType !== AccountType.SUBSCRIPTION) {
      return res.status(400).json({
        success: false,
        message: '只有订阅号支持识别码绑定'
      });
    }

    // 解析识别码（格式：工号+手机号后4位）
    const match = content.trim().match(/^(EMP\d+)(\d{4})$/);

    if (!match) {
      return res.json({
        success: false,
        message: '识别码格式不正确',
        hint: '请输入：工号+手机号后4位，例如：EMP0011234'
      });
    }

    const [_, employeeId, phoneSuffix] = match;

    // 验证员工信息
    const [employees] = await pool.query(
      'SELECT * FROM employees WHERE employee_id = ? AND phone LIKE ?',
      [employeeId, `%${phoneSuffix}`]
    );

    if (employees.length === 0) {
      return res.json({
        success: false,
        message: '工号或手机号后4位不正确',
        hint: '请检查工号和手机号后4位'
      });
    }

    const employee = employees[0];

    // 检查是否已绑定
    const [existing] = await pool.query(
      'SELECT * FROM employee_bindings WHERE employee_id = ? AND account_id = ?',
      [employeeId, account_id]
    );

    if (existing.length > 0) {
      return res.json({
        success: false,
        message: '您已绑定此公众号'
      });
    }

    // 绑定员工
    await pool.query(
      `INSERT INTO employee_bindings (employee_id, account_id, openid, is_verified, verified_at, created_at)
       VALUES (?, ?, ?, TRUE, NOW(), NOW())`,
      [employeeId, account_id, openid]
    );

    logger.info('识别码绑定成功', { employeeId, openid, accountId });

    res.json({
      success: true,
      message: '绑定成功！',
      data: {
        employee_id: employee.employee_id,
        employee_name: employee.name,
        bind_method: 'verification_code'
      }
    });

  } catch (error: any) {
    logger.error('识别码验证失败', error);
    res.status(500).json({
      success: false,
      message: error.message || '验证失败'
    });
  }
});

export default router;
```

### 4.2 统一员工绑定页面

```tsx
// employee-h5/src/pages/EmployeeUnifiedBindPage.tsx

import { useState, useEffect } from 'react';
import { WechatAuth } from '@/components/WechatAuth';
import { Card, CardContent, CardDescription, CardHeader, CardTitle } from '@/components/ui/card';
import { Button } from '@/components/ui/button';
import { Input } from '@/components/ui/input';
import { Label } from '@/components/ui/label';
import { Alert, AlertDescription } from '@/components/ui/alert';
import { Badge } from '@/components/ui/badge';
import { Loader2, UserCheck, AlertCircle, Smartphone, MessageSquare } from 'lucide-react';

interface AccountConfig {
  account_id: number;
  account_name: string;
  account_type: 'subscription' | 'service';
  bind_method: 'web_auth' | 'verification_code';
  bind_method_description: string;
  is_web_auth_supported: boolean;
}

export function EmployeeUnifiedBindPage() {
  const [accountId, setAccountId] = useState<number | null>(null);
  const [accountConfig, setAccountConfig] = useState<AccountConfig | null>(null);
  const [loading, setLoading] = useState(true);

  const [step, setStep] = useState<'config' | 'auth' | 'verify' | 'success' | 'error'>('config');
  const [openid, setOpenid] = useState<string>('');
  const [userInfo, setUserInfo] = useState<any>(null);

  const [formData, setFormData] = useState({
    employee_id: '',
    phone_suffix: '',
  });

  const [error, setError] = useState('');

  // 1. 获取公众号绑定配置
  useEffect(() => {
    const fetchConfig = async () => {
      try {
        // 从URL获取account_id
        const urlParams = new URLSearchParams(window.location.search);
        const id = urlParams.get('account_id');

        if (!id) {
          setError('缺少公众号ID');
          setLoading(false);
          return;
        }

        setAccountId(parseInt(id));

        // 获取绑定配置
        const response = await fetch(`/api/employee/bind-config?account_id=${id}`);
        const result = await response.json();

        if (!result.success) {
          throw new Error(result.message);
        }

        setAccountConfig(result.data);
        setLoading(false);

        // 如果是服务号，自动进入授权步骤
        if (result.data.is_web_auth_supported) {
          setStep('auth');
        } else {
          // 订阅号，直接进入验证步骤
          setStep('verify');
        }

      } catch (error: any) {
        setError(error.message);
        setLoading(false);
      }
    };

    fetchConfig();
  }, []);

  // 2. H5授权成功回调（服务号）
  const handleAuthSuccess = (fetchedOpenid: string, fetchedUserInfo?: any) => {
    setOpenid(fetchedOpenid);
    setUserInfo(fetchedUserInfo);
    setStep('verify');
  };

  // 3. 提交绑定
  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    setError('');

    if (!formData.employee_id || !formData.phone_suffix) {
      setError('请填写完整信息');
      return;
    }

    setLoading(true);

    try {
      const response = await fetch('/api/employee/bind', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        credentials: 'include',
        body: JSON.stringify({
          account_id: accountId,
          employee_id: formData.employee_id,
          phone_suffix: formData.phone_suffix,
          openid: openid, // 服务号有openid，订阅号为空
          bind_method: accountConfig?.bind_method,
        }),
      });

      const result = await response.json();

      if (!result.success) {
        throw new Error(result.message);
      }

      setStep('success');
      localStorage.setItem('employee', JSON.stringify(result.data));

    } catch (error: any) {
      setError(error.message);
      setLoading(false);
    }
  };

  if (loading) {
    return <div className="flex items-center justify-center min-h-screen"><Loader2 className="animate-spin" /></div>;
  }

  if (!accountConfig) {
    return <div className="p-4 text-center text-red-500">{error || '加载失败'}</div>;
  }

  return (
    <div className="min-h-screen bg-gradient-to-br from-green-50 to-blue-50 p-4">
      <div className="max-w-md mx-auto">
        {/* 公众号信息卡片 */}
        <Card className="mt-8">
          <CardHeader>
            <div className="flex items-center justify-between">
              <CardTitle>{accountConfig.account_name}</CardTitle>
              <Badge variant={accountConfig.account_type === 'service' ? 'default' : 'secondary'}>
                {accountConfig.account_type === 'service' ? '服务号' : '订阅号'}
              </Badge>
            </div>
            <CardDescription>
              {accountConfig.bind_method_description}
            </CardDescription>
          </CardHeader>
        </Card>

        {/* 步骤1：H5授权（仅服务号） */}
        {step === 'auth' && accountConfig.is_web_auth_supported && (
          <div className="mt-4">
            <WechatAuth
              onAuthSuccess={handleAuthSuccess}
              onAuthError={(err) => setError(err)}
              scope="snsapi_base"
            />
          </div>
        )}

        {/* 步骤2：验证员工信息 */}
        {step === 'verify' && (
          <Card className="mt-4">
            <CardHeader>
              <div className="flex items-center justify-center mb-4">
                <div className="w-16 h-16 bg-blue-100 rounded-full flex items-center justify-center">
                  {accountConfig.account_type === 'service' ? (
                    <Smartphone className="w-8 h-8 text-blue-600" />
                  ) : (
                    <MessageSquare className="w-8 h-8 text-blue-600" />
                  )}
                </div>
              </div>
              <CardTitle className="text-center">员工身份验证</CardTitle>
              <CardDescription className="text-center">
                {accountConfig.account_type === 'service'
                  ? '请输入您的工号和手机号后4位，完成身份验证'
                  : '请先在公众号对话框发送识别码，然后输入工号和手机号后4位验证'}
              </CardDescription>
            </CardHeader>
            <CardContent>
              {/* 订阅号提示 */}
              {accountConfig.account_type === 'subscription' && (
                <Alert className="mb-4">
                  <MessageSquare className="h-4 w-4" />
                  <AlertDescription>
                    订阅号需要先在公众号对话框发送识别码（格式：工号+手机号后4位，如EMP0011234）
                  </AlertDescription>
                </Alert>
              )}

              {/* 错误提示 */}
              {error && (
                <Alert variant="destructive" className="mb-4">
                  <AlertCircle className="h-4 w-4" />
                  <AlertDescription>{error}</AlertDescription>
                </Alert>
              )}

              <form onSubmit={handleSubmit} className="space-y-4">
                <div className="space-y-2">
                  <Label htmlFor="employee_id">工号 *</Label>
                  <Input
                    id="employee_id"
                    placeholder="如：EMP001"
                    value={formData.employee_id}
                    onChange={(e) =>
                      setFormData({ ...formData, employee_id: e.target.value.toUpperCase() })
                    }
                    disabled={loading}
                    className="uppercase"
                  />
                </div>

                <div className="space-y-2">
                  <Label htmlFor="phone_suffix">手机号后4位 *</Label>
                  <Input
                    id="phone_suffix"
                    placeholder="如：1234"
                    value={formData.phone_suffix}
                    onChange={(e) => {
                      const value = e.target.value.replace(/\D/g, '').slice(0, 4);
                      setFormData({ ...formData, phone_suffix: value });
                    }}
                    disabled={loading}
                    maxLength={4}
                    type="tel"
                  />
                </div>

                <Button type="submit" className="w-full" disabled={loading}>
                  {loading ? (
                    <>
                      <Loader2 className="mr-2 h-4 w-4 animate-spin" />
                      验证中...
                    </>
                  ) : (
                    '验证并绑定'
                  )}
                </Button>
              </form>
            </CardContent>
          </Card>
        )}

        {/* 步骤3：绑定成功 */}
        {step === 'success' && (
          <Card className="mt-4 border-green-200">
            <CardHeader className="text-center">
              <div className="flex items-center justify-center mb-4">
                <div className="w-16 h-16 bg-green-100 rounded-full flex items-center justify-center">
                  <UserCheck className="w-8 h-8 text-green-600" />
                </div>
              </div>
              <CardTitle className="text-green-600">绑定成功！</CardTitle>
              <CardDescription>
                恭喜您，已成功绑定员工身份
              </CardDescription>
            </CardHeader>
            <CardContent className="space-y-4">
              <div className="bg-green-50 p-4 rounded-lg">
                <p className="text-sm text-green-800">
                  {accountConfig.account_type === 'service'
                    ? '您的专属推广二维码已生成，现在可以开始推广公众号了！'
                    : '绑定成功！您的推广二维码正在生成中，请稍后查看。'}
                </p>
              </div>

              <Button onClick={() => window.location.href = '/my-promotion'} className="w-full">
                查看我的推广码
              </Button>
            </CardContent>
          </Card>
        )}
      </div>
    </div>
  );
}
```

### 5.1 管理员配置

1. **添加公众号**
   - 选择公众号类型（订阅号/服务号）
   - 填写公众号信息
   - 系统自动识别可用功能

2. **生成绑定链接**
   - 服务号：`https://your-domain.com/bind?account_id=1`
   - 订阅号：`https://your-domain.com/bind?account_id=2`（但需要先发送识别码）

### 5.2 员工绑定流程

#### 服务号流程

```
1. 员工打开绑定链接
   ↓
2. 系统自动获取OpenID（静默授权）
   ↓
3. 员工输入工号+手机号后4位
   ↓
4. 验证成功，完成绑定
```

#### 订阅号流程

```
1. 员工在公众号对话框发送识别码（EMP0011234）
   ↓
2. 系统验证识别码，获取OpenID
   ↓
3. 员工打开绑定链接
   ↓
4. 输入工号+手机号后4位
   ↓
5. 验证成功，完成绑定
```

## 六、核心优势

✅ **统一入口**：员工通过同一个H5页面绑定，无需区分公众号类型
✅ **自动识别**：系统根据公众号类型自动选择合适的绑定方式
✅ **用户友好**：服务号无感知，订阅号有明确指引
✅ **灵活切换**：支持同时管理多个不同类型的公众号
✅ **可扩展性**：未来可以轻松添加新的公众号类型或绑定方式
