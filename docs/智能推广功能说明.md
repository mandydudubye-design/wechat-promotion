# 智能推广功能说明

## 功能概述

系统现在支持根据公众号类型**自动选择推广模式**，为不同类型的公众号提供最优的推广方案。

## 推广模式

### 1. 二维码模式（QR_CODE）
**适用公众号**：服务号、已认证订阅号

**特点**：
- ✅ 使用微信高级接口生成带参数的二维码
- ✅ 用户扫码关注后自动关联员工
- ✅ 无需用户手动操作
- ✅ 体验最佳，转化率最高

**流程**：
1. 员工在H5端选择要推广的公众号
2. 系统自动生成专属二维码（包含员工ID）
3. 用户扫码关注公众号
4. 微信回调服务器，自动建立员工-用户关联关系
5. 推广数据自动记录

### 2. 验证码模式（VERIFICATION_CODE）
**适用公众号**：未认证订阅号

**特点**：
- ❌ 无微信高级接口权限
- ✅ 使用验证码作为关联方式
- ⚠️ 需要用户手动操作
- ⚠️ 体验相对较差，但仍有可用性

**流程**：
1. 员工在H5端选择要推广的公众号
2. 系统生成6位验证码
3. 员工分享推广链接给用户
4. 用户关注公众号
5. 用户向公众号回复验证码
6. 系统验证验证码，建立关联关系
7. 推广数据自动记录

## API 使用

### 1. 生成员工推广码（智能选择模式）

**接口**：`POST /api/smart-promotion/employee-code`

**请求参数**：
```json
{
  "employeeId": "EMP001",
  "employeeName": "张三",
  "accountId": 1
}
```

**响应示例**（二维码模式）：
```json
{
  "code": 200,
  "message": "生成成功",
  "timestamp": 1234567890,
  "data": {
    "mode": "qr_code",
    "modeLabel": "二维码模式",
    "message": "二维码模式：扫码即可关注并自动关联",
    "qrCodeUrl": "https://mp.weixin.qq.com/cgi-bin/showqrcode?ticket=xxx",
    "qrCodeExpireSeconds": 2592000
  }
}
```

**响应示例**（验证码模式）：
```json
{
  "code": 200,
  "message": "生成成功",
  "timestamp": 1234567890,
  "data": {
    "mode": "verification_code",
    "modeLabel": "验证码模式",
    "message": "验证码模式：用户关注后需回复验证码",
    "verificationCode": "123456"
  }
}
```

### 2. 获取公众号推广配置

**接口**：`GET /api/smart-promotion/config/:accountId`

**响应示例**：
```json
{
  "code": 200,
  "message": "获取成功",
  "timestamp": 1234567890,
  "data": {
    "accountId": 1,
    "accountName": "企业官方号",
    "accountType": "service",
    "verified": true,
    "promotionMode": "qr_code",
    "modeLabel": "二维码模式",
    "canGeneratePersonalQr": true
  }
}
```

## 技术实现

### 核心服务：promotionService.ts

```typescript
// 根据公众号类型判断推广模式
export function determinePromotionMode(
  accountType: 'service' | 'subscription',
  verified: boolean
): PromotionMode {
  // 服务号或已认证订阅号：使用二维码模式
  if (accountType === 'service' || verified) {
    return PromotionMode.QR_CODE;
  }
  // 未认证订阅号：使用验证码模式
  return PromotionMode.VERIFICATION_CODE;
}
```

### 新增路由：smart-promotion.ts

提供了智能推广相关的API端点：
- `POST /api/smart-promotion/employee-code` - 智能生成推广码
- `GET /api/smart-promotion/config/:accountId` - 获取推广配置

## 注意事项

1. **微信API配置**：使用二维码模式需要配置微信API密钥（WECHAT_APP_ID、WECHAT_APP_SECRET）
2. **数据库字段**：已在employees表中添加verification_code字段
3. **模式切换**：系统会根据公众号类型自动选择最适合的推广模式
4. **降级处理**：如果微信API不可用，系统会自动降级到模拟模式
