**修复内容**：
1. `smart-promotion.ts`：添加 `safeScanTime` 转换逻辑，将 ISO 日期转为 MySQL 兼容格式
2. `smart-promotion.ts`：IP 地址截断到 50 字符防止超出 `ip_address varchar(50)` 字段限制
3. `smart-promotion.ts`：catch 块增加 `error.stack` 日志便于排查

## 实现订阅号关注追踪方案（推广码模式）

**方案背景**：已认证订阅号无法生成带参数二维码，改用「用户在公众号对话框回复推广码（= 员工工号）」来实现关注归因。

**改造内容**：

1. **`wechat.ts`** — 核心改造：

3. **`app.ts`** — 启动时自动创建 `follow_records` 表 + 确保 `employees` 表有 `promotion_count` 字段
